Политика касаемо обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных определяет порядок и условия обработки персональных данных, предпринимаемые компанией Xenon-Matrix (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных.

2. Основные понятия

  • Веб-сайт — ресурсы, доступные по адресу Xenon-Matrix.blog.
  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся к Пользователю.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Фамилия, имя, отчество;
  • Адрес электронной почты (Email);
  • Номера телефонов;
  • Обезличенные данные (cookies) с помощью сервисов интернет-статистики.

4. Цели обработки персональных данных

  • Идентификация Пользователя.
  • Установление обратной связи.
  • Информирование Пользователя. Вы всегда можете отказаться от рассылки, написав на hello@xenon-matrix.blog.

5. Безопасность данных

  • Оператор принимает все меры для защиты данных от неправомерного доступа.
  • Данные никогда не передаются третьим лицам, кроме случаев, предусмотренных законом.

6. Актуализация данных

Пользователь может в любой момент актуализировать свои персональные данные, направив уведомление Оператору на электронный адрес: hello@xenon-matrix.blog.

7. Контактная информация

Пользователь может получить любые разъяснения по вопросам обработки данных, обратившись к Оператору по электронной почте: hello@xenon-matrix.blog.
